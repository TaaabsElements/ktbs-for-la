# ktbs-for-la

`<ktbs-for-la>` is a web app using some `taaabs` components. It helps learning analysts work with a ktbs.

Example:

    <ktbs-for-la ktbs-url="https://YOUR_KTBS_URL/"></ktbs-for-la>
